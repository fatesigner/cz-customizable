## Install

```bash
# Make sure the corresponding version for commitizen is installed
npm i -D commitizen

npm i -D @fatesigner/cz-customizable
```

## Usage
Add this config to package.json

```bash
...
"config": {
  "commitizen": {
    "path": "node_modules/@fatesigner/cz-customizable"
  }
}
```
